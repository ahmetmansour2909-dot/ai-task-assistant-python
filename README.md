# AI Task Assistant 

AI Task Assistant is a simple **command-line intelligent task management system** developed using **Python**.  
The system acts as a rule-based AI agent that helps users organize daily tasks by automatically **classifying tasks**, **assigning priorities**, and **providing AI explanations** based on natural language input.

##  Features
- Add new tasks using **natural language input**
- Automatically assign task **priority** (High / Medium / Low)
- Automatically detect task **category** (Study, Work, Personal, General)
- Store tasks persistently using a **JSON file**
- Edit and delete existing tasks
- Display AI-generated explanations for decisions
- Show task summaries and AI recommendations
- Simple and user-friendly **command-line menu**

---

##  AI Agent Logic
The AI Task Assistant uses a **rule-based intelligent agent** that analyzes task descriptions using keywords such as:
- **Priority keywords:** exam, project, meeting, deadline
- **Time keywords:** today, tomorrow, urgent
- **Category keywords:** study, work, personal

Based on these rules, the agent:
- Assigns priority dynamically
- Determines task category
- Explains the decision in a human-readable way
