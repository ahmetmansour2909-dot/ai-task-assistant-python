# AI Task Assistant

AI Task Assistant is a command-line based task management application developed using Python.
The system implements a rule-based intelligent agent that analyzes user input, categorizes tasks,
assigns priorities, and provides AI-driven suggestions.

## Features
- Add, edit, and delete tasks
- Intelligent task categorization and prioritization
- AI-generated task suggestions
- Task summary and workload analysis
- Persistent data storage using JSON

## Technologies
- Python 3
- JSON for data storage

## How to Run
1. Navigate to the project directory
2. Run the following command:
   python main.py

Command Line Application
